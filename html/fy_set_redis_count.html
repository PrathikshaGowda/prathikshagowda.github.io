fy_set_redis_count\uf0c1

These are the functions which are common among the REST APIs.

fy_set_redis_count.incr_redis_field(route, localMemory=None, fyId=None, userFlag=False)[source]\uf0c1
A Function which update redis with number of times an API is called and also number of hits on API by perticular fyersID.
Parameters:
  • localMemory - End point of redis cluster.
  • fyId - fyersID of the client is passed to get the API hit count of perticular client.
  • fyId - None to get the API count.
  • userFlag - True to get the API hit count for perticular client.
  • userFlag - False to get the API count.
  • route - route of API.

Return:

Success :  localMemory.hincrby(routeCountKey,route) -->> 12550429L
